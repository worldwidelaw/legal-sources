# ZA/SARB-Notices — South African Reserve Bank Regulatory Publications

Prudential Authority regulatory doctrine from the South African Reserve Bank:
Banks Directives, Banks Circulars, Guidance Notes, PA Communications,
Insurance Act Notices, and Consultation Documents.

## Data Access

Publication detail pages are discovered via the sitemap at
`resbank.co.za/sitemap.xml`. Each page links to one or more PDFs hosted
at `/content/dam/sarb/publications/`. Text is extracted from PDFs using
`common/pdf_extract`.
