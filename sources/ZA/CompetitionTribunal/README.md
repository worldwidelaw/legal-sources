# ZA/CompetitionTribunal — Competition Tribunal of South Africa

Decided cases from the Competition Tribunal of South Africa, including merger
approvals, prohibited practice cases, consent orders, and exemption applications.

**Website:** https://www.comptrib.co.za
**Data type:** Case law
**Language:** English
**Coverage:** All decided cases (~3000+)

## Data Source

Cases are scraped from the tribunal's decided-cases listing, which is paginated
HTML. Full text is extracted from PDF order and reasons documents.
