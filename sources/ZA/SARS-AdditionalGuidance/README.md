# ZA/SARS-AdditionalGuidance — South Africa SARS Practice Notes & Guides

Practice Notes, Comprehensive Guides, and External Guides from the South African Revenue Service (SARS).

## Coverage

- **Practice Notes** (~114): Historical and current tax practice guidance (income tax, retirement funds, etc.)
- **Comprehensive Guides** (~8): Detailed guides on CGT, dividends tax, trusts, individual returns, STC
- **External Guides** (~161): Taxpayer-facing guides on eFiling, payments, registration, returns

## Relationship to ZA/SARS-Interpretations

This source complements `ZA/SARS-Interpretations`, which covers Interpretation Notes and Binding Rulings (BGR/BCR/BPR/VR/VCR). There is no overlap.

## Data Access

Documents are discovered via the WordPress REST API (`/wp-json/wp/v2/media`) and downloaded as PDFs. Full text is extracted using the project's common PDF extraction pipeline.
