# MX/CNBV — Comisión Nacional Bancaria y de Valores

Regulatory provisions, circulars, laws, and agreements from Mexico's National
Banking and Securities Commission (CNBV). The CNBV supervises banking
institutions, securities markets, fintech companies, and payment systems.

## Data

- ~112 consolidated regulatory PDF documents
- Full text extracted via `pdf_extract`
- Language: Spanish
- Coverage: banking, securities, fintech, payments, anti-money laundering

## Access Method

- SharePoint REST API at `cnbv.gob.mx/_api/web/GetFolderByServerRelativeUrl('/Normatividad')/Files`
- Direct PDF download from `cnbv.gob.mx/Normatividad/`
- SSL verification disabled (site certificate issues)
