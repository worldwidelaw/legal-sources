# San Marino Tribunal Decisions

**Source:** [https://www.tribunale.sm/](https://www.tribunale.sm/)
**Country:** SM
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `auth_required`

**Details:** Case law portal (giurisprudenzarsm.org) requires LEGALE profile authentication restricted to judges, magistrates, and registered attorneys. No public access to court decisions. Tribunal website (tribunale.sm) has no case database or API.

## How you can help

This source requires login credentials or API keys.
- If this data is publicly available through another channel, please file an issue
- If you have legitimate access, you could contribute a data dump

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
