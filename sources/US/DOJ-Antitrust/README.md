# US/DOJ-Antitrust — U.S. DOJ Antitrust Division Case Filings

Antitrust case filings from the U.S. Department of Justice Antitrust Division.
Covers civil and criminal antitrust enforcement — complaints, final judgments,
competitive impact statements, plea agreements, and consent decrees.

**Website:** https://www.justice.gov/atr
**Data type:** Case law
**Language:** English
**Coverage:** ~923 antitrust cases (price fixing, monopolization, mergers)

## Data Source

Cases are scraped from the alphabetical case listing. Each case detail page
provides structured metadata (case type, violations, dates) and links to
PDF documents. Full text is extracted from the primary document PDF.
