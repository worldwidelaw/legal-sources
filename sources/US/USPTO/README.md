# USPTO Patent Trial and Appeal Board Decisions

**Source:** [https://data.uspto.gov/](https://data.uspto.gov/)
**Country:** US
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** API key required

**Technical reason:** `api_key_required`

**Details:** From issue #290: PTAB API moved to ODP v3 at api.uspto.gov. Full text available via documentOCRText field. Requires free ODP API key from account.uspto.gov/api-manager/ — admin needs to register.

## How you can help

The API requires a key that must be obtained from the provider.
- If you have a key or know how to get one freely, please let us know
