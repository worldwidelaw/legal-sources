# US/DC-Code — District of Columbia Code

The official DC Code, sourced from the [DCCouncil/law-xml](https://github.com/DCCouncil/law-xml) GitHub repository.

## Coverage

- **Scope:** All 53 titles of the District of Columbia Code
- **Sections:** ~21,163 individual code sections
- **Format:** Structured XML with text, annotations, and legislative history
- **Language:** English
- **Updates:** Actively maintained by the DC Council

## Data Access

Uses the GitHub API to discover section XML files, then fetches raw XML content
from `raw.githubusercontent.com`. Each section is a self-contained XML file with
full text, paragraph structure, and annotations.
