# US/DE-Code — Delaware Code

The official Delaware Code from [delcode.delaware.gov](https://delcode.delaware.gov).

## Coverage

- **Scope:** All 31 titles of the Delaware Code
- **Format:** HTML pages with section-level content
- **Language:** English
- **Notable:** Title 8 (General Corporation Law) — the foundation of US corporate law

## Data Access

Crawls the official Delaware Code Online website. Discovers all titles, chapters,
and subchapters, then parses individual sections from HTML pages using
`div.Section` / `div.SectionHead` structure.
