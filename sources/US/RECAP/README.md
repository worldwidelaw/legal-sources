# RECAP Archive - Federal Court Documents

**Source:** [https://www.courtlistener.com/recap/](https://www.courtlistener.com/recap/)
**Country:** US
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** API key required

**Technical reason:** `needs_token`

**Details:** RECAP uses CourtListener API (courtlistener.com/api/rest/v4/recap/) which requires authentication token. Same blocker as US/CourtListener. Admin needs to register at courtlistener.com and provide COURTLISTENER_API_TOKEN.

## How you can help

The API requires a key that must be obtained from the provider.
- If you have a key or know how to get one freely, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
