# Virginia Legislative Information System (LIS) API

**Source:** [https://lis.virginia.gov/developers](https://lis.virginia.gov/developers)
**Country:** US
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** API key required

**Technical reason:** `api_key_required`

**Details:** LIS API requires partner API key (Bearer token). Admin must register at lis.virginia.gov/apiregistration. Legacy CGI endpoint decommissioned. No public bill text endpoint found.

## How you can help

The API requires a key that must be obtained from the provider.
- If you have a key or know how to get one freely, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
