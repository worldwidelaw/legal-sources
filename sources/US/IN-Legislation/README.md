# Indiana Code (iga.in.gov)

**Source:** [https://iga.in.gov/laws/current/ic/](https://iga.in.gov/laws/current/ic/)
**Country:** US
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** Registration required

**Technical reason:** `requires_api_key_registration`

**Details:** Issue #500: iga.in.gov is now an entirely SPA-driven site — the per-title PDF URLs (iga.in.gov/ic/YYYY/Title_N.pdf) are replaced by a React shell that returns text/html for all /ic/ paths. The MyIGA Hypermedia API (docs.api.iga.in.gov) requires an admin-registered free API token via apitoken.request@iga.in.gov. Requires admin action to obtain an API key before this source can be rebuilt.

## How you can help

Access requires creating an account or registering for an API key.
- If the registration is free and the data is open, we may revisit this

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
