**Status:** Blocked

## Why this source is blocked

**Category:** API key required

**Technical reason:** `needs_token`

**Details:** Redirects to CourtListener API/bulk data (needs auth token). US Supreme Court cases already covered by US/CaselawAccessProject. Same blocker as US/CourtListener.

## How you can help

The API requires a key that must be obtained from the provider.
- If you have a key or know how to get one freely, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
