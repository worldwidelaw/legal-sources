# US International Trade Commission Decisions

**Source:** [https://edis.usitc.gov/](https://edis.usitc.gov/)
**Country:** US
**Data types:** case_law, doctrine
**Status:** Blocked

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `auth_required`

**Details:** EDIS API at edis.usitc.gov/data/ provides metadata anonymously but PDF downloads require Login.gov auth + Bearer token. No plain-text endpoint exists.
