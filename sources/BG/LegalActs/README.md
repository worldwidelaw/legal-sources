# Bulgarian Unified Court Decisions Portal (УПЕП)

**Source:** [http://legalacts.justice.bg](http://legalacts.justice.bg)
**Country:** BG
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** CAPTCHA protection

**Technical reason:** `captcha_protected`

**Details:** BLOCKED: All search endpoints require CAPTCHA. No REST API, no bulk listing, no enumeration endpoint. Full text accessible via ECLI at /GetActContent/{ECLI} but no way to discover ECLIs programmatically without CAPTCHA-protected search.

## How you can help

The site requires solving CAPTCHAs to access content.
- If you know of an alternative API or bulk download for this data, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
