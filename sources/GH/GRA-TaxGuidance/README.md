# GH/GRA-TaxGuidance — Ghana Revenue Authority Practice Notes & Guidelines

Practice notes, tax guidelines, advance ruling guidance, legislation texts, and compliance materials from the Ghana Revenue Authority (GRA).

## Coverage

- **Practice Notes** (~23): Tax administration guidance on WHT, VAT, income tax
- **Legislation** (~47): Acts, amendments, regulations (Income Tax Act, VAT Act, Revenue Administration Act, etc.)
- **Guides** (~13): Taxpayer guides, compliance guides, CRS reporting guides
- **Advance Rulings** (~3): Advance ruling instructions and guidance
- **Other** (~96): Compliance plans, forms, service charters, tax codes

## Data Access

Documents are discovered via the WordPress REST API (`/wp-json/wp/v2/media`) and downloaded as PDFs. Full text is extracted using the project's common PDF extraction pipeline.
