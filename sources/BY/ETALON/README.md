**Source:** [https://etalonline.by](https://etalonline.by)
**Country:** BY
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** js_rendering_required

**Technical reason:** `js_rendering_required`

**Details:** Issue #248: 26 codified laws only. Search for additional documents requires JS rendering (search results not in HTML). Site has severe bandwidth throttling (~2KB/s). Expanding beyond codes requires browser automation.
