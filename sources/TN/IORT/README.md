# Tunisia Official Printing and Publishing Office (IORT)

**Source:** [https://www.iort.gov.tn/](https://www.iort.gov.tn/)
**Country:** TN
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** JavaScript SPA (single-page application)

**Technical reason:** `no_api_access`

**Details:** iort.gov.tn returns 403. legislation.tn times out. 9anoun.tn uses Livewire dynamic loading (needs Playwright). No programmatic access to Tunisian legislation full text without browser automation.
