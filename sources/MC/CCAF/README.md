# MC/CCAF — Monaco Commission for Financial Activities Control

Sanction decisions, warnings about unauthorized entities, cooperation agreements,
and regulatory communications from Monaco's financial regulator (CCAF).

## Data

- **Type:** doctrine
- **Documents:** ~32 publications (2022–2026)
- **Language:** English (via WPML translations)
- **Coverage:** AML/CFT sanctions, unauthorized entity warnings, IOSCO/IFREFI participation, regulatory updates

## Method

WordPress REST API (`/wp-json/wp/v2/posts?lang=en`). PDF extraction for sanction
decisions linked from post content.
