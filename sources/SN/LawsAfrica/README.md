# Senegal Legislation via Laws.Africa

**Source:** [https://api.laws.africa/v3/akn/sn/](https://api.laws.africa/v3/akn/sn/)
**Country:** SN
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** API key required

**Technical reason:** `requires_api_token`

**Details:** REST API at api.laws.africa/v3/akn/sn/. Structured full text in Akoma Ntoso XML. Requires free registration token at edit.laws.africa/accounts/login/. Admin needs to sign up and provide LAWS_AFRICA_TOKEN.

## How you can help

The API requires a key that must be obtained from the provider.
- If you have a key or know how to get one freely, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
