# Laws.Africa Content API (African Legislation)

**Source:** [https://laws.africa/api/](https://laws.africa/api/)
**Country:** INTL
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** API key required

**Technical reason:** `requires_api_key_signup`

**Details:** All API endpoints return 403 without auth token. Requires free account signup at edit.laws.africa. National legislation also needs paid per-country subscription. Admin needs to sign up at https://edit.laws.africa/accounts/login/ and get token from profile/api page.

## How you can help

The API requires a key that must be obtained from the provider.
- If you have a key or know how to get one freely, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
