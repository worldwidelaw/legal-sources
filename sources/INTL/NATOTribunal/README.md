# INTL/NATOTribunal — NATO Administrative Tribunal

**Source:** [NATO Administrative Tribunal](https://www.nato.int/cps/en/natohq/topics_114072.htm)

Judgments and orders of the NATO Administrative Tribunal, the judicial body
that resolves employment disputes between NATO and its staff. Annual PDF
compilations from 2013 to present.

## Data

- **Type:** case_law (judgments and orders)
- **Coverage:** 2013-present, ~150+ decisions
- **Format:** Annual PDF compilations
- **Language:** English

## Strategy

1. Download 13 annual PDF compilations (2013-2025)
2. Extract text and split into individual judgments/orders using ID patterns
3. Parse metadata (date, case number, keywords) from each decision header
