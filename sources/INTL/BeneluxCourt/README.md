# INTL/BeneluxCourt — Benelux Court of Justice

Judgments of the Benelux Court of Justice (Cour de Justice Benelux / Benelux Gerechtshof).

Covers preliminary rulings, BOIP intellectual property appeals, and civil service disputes since 1974. ~370 judgments in French and Dutch.

## Data Source

- **Website:** https://www.courbeneluxhof.int/
- **Listing:** https://www.courbeneluxhof.int/fr/arrets-conclusions/
- **Format:** HTML index → individual case pages → PDF downloads
- **Languages:** French, Dutch
