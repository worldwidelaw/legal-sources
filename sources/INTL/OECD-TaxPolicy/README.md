## Why this source is blocked

**Category:** website_blocks_automated_access

**Technical reason:** `website_blocks_automated_access`

**Details:** OECD blocks all programmatic access (403 on HTML pages, search, RSS, sitemaps, APIs). PDFs downloadable only with undiscoverable CMS-internal hash in URL path.
