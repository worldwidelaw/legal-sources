**Country:** INTL
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** Scanned PDFs requiring OCR

**Technical reason:** `scanned_pdfs_no_ocr`

**Details:** 9/10 biblio.ohada.org PDFs are scanned images (JBIG2) without text layers. ohada.org English translation PDF links are dead (404). Only accounting law (AUDCIF) has extractable text. Requires OCR to unblock.
