# TR/RekabetKurumu — Turkish Competition Authority

Board decisions of the Turkish Competition Authority (Rekabet Kurumu), covering mergers & acquisitions, competition violations, negative assessments, exemptions, and privatization decisions.

## Data Source

- **Website:** https://www.rekabet.gov.tr/tr/Kararlar
- **Format:** Server-rendered HTML search + PDF decisions
- **Auth:** None required
- **Coverage:** ~10,200+ board decisions since 1997

## Data Types

- `case_law` — competition board decisions
