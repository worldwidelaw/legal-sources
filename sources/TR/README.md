# Turkey — Legal Data Sources

> **Last updated:** 2026-02-21

## Overview

Turkey is a **civil law** country (influenced by Swiss civil code), **unitary**. Large jurisdiction with significant legal output. Provides **consolidated legislation** through **Mevzuat** (mevzuat.gov.tr).

## Consolidated Legislation vs. Official Journal

**Consolidated** via Mevzuat (mevzuat.gov.tr). Resmi Gazete is the official gazette.

## Sub-jurisdictions

**Unitary state** — 81 provinces with no legislative autonomy.
