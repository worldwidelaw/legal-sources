# Knesset OData API (Legislation)

**Source:** [https://knesset.gov.il/Odata/ParliamentInfo.svc](https://knesset.gov.il/Odata/ParliamentInfo.svc)
**Country:** IL
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** Geographic restriction

**Technical reason:** `geo_blocking`

**Details:** From issue #219. Knesset OData API with KNS_Bill, laws tables. API works from Israeli IPs only. Returns maintenance page from US/EU. Requires VPN/proxy with Israeli IP. hasadna/knesset-data GitHub projects exist but also need Israeli IP.

## How you can help

The site blocks access from non-local IP addresses.
- A proxy or VPN in the target country could work
- If you're located in this jurisdiction, you could help test access

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
