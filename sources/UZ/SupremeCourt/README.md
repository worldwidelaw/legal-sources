## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `requires_national_id_auth`

**Details:** Duplicate of UZ/SupremeCourt-Decisions. Court decisions at publication.sud.uz require X-AUTH-TOKEN obtained via Uzbekistan One ID national identity login. No public API for decisions.
