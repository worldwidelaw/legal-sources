# Uzbekistan Supreme Court Decisions

**Source:** [https://supremecourt.uz/](https://supremecourt.uz/)
**Country:** UZ
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** requires_headless_browser

**Technical reason:** `requires_headless_browser`

**Details:** public.sud.uz is Angular SPA (no discoverable API). publication.sud.uz requires token auth. adolatapi1.sud.uz only has /api/public/guides endpoints. Court decisions not accessible without headless browser.
