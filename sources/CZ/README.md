# Czech Republic — Legal Data Sources

> **Last updated:** 2026-02-21

## Overview

The Czech Republic is a **civil law** country, **unitary**. Launched **e-Sbirka** in January 2024 — a modern consolidated legislation system (CC BY 4.0, 43K+ acts). NALUS database provides 70K+ Constitutional Court decisions since 1993.
