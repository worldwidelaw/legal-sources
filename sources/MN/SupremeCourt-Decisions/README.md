## Why this source is blocked

**Category:** Cloudflare protection

**Technical reason:** `cloudflare_js_rendering`

**Details:** Court decisions at shuukh.mn are loaded via JavaScript behind Cloudflare. No public API. Search results require browser automation. supremecourt.mn only has summaries, not full text decisions.
