# CN/SAFE — State Administration of Foreign Exchange Regulations

English translations of Chinese foreign exchange regulations from SAFE
(www.safe.gov.cn/en/). Covers foreign exchange administration, cross-border
payments, capital accounts, and currency management (~128 documents).

## Data Source

- **Portal**: https://www.safe.gov.cn/en/RulesandRegulations/index.html
- **Format**: HTML scraping with pagination (7 pages)
- **Language**: English (translations of Chinese originals)
- **Coverage**: Foreign exchange regulations, circulars, notices
