# CN/STA-TaxPolicy — State Taxation Administration Tax Policy Database

Chinese tax policy documents from the State Taxation Administration (STA) at
fgk.chinatax.gov.cn. Covers tax laws, administrative regulations, normative
documents, finance/tax documents, and work notices (~4,977 documents).

## Data Source

- **Portal**: https://fgk.chinatax.gov.cn/zcfgk/c100006/listflfg.html
- **API**: POST `https://www.chinatax.gov.cn/getFileListByCodeId`
- **Format**: JSON listing API + HTML content pages
- **Language**: Chinese (zh)
- **Coverage**: Tax laws, regulations, normative documents, policy notices
- **Update frequency**: Ongoing (new policies added regularly)
