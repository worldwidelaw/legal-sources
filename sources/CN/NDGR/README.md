**Status:** Blocked

## Why this source is blocked

**Category:** api_returns_empty_results

**Technical reason:** `api_returns_empty_results`

**Details:** Search API at sousuo.www.gov.cn returns facet counts (12,678 regulations) but 0 actual results regardless of query parameters. Page at gov.cn/zhengce/xzfg/ returns 404. Original URL also 404. Requires browser JS execution or undocumented API params.
