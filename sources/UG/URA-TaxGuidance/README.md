# UG/URA-TaxGuidance — Uganda Revenue Authority Tax Guidance

Tax guidance documents, case digests, sector guides, and compliance materials from the Uganda Revenue Authority (URA).

## Coverage

- **Tax Guides** (~100+): Sector-specific guides (agriculture, mining, real estate, etc.)
- **Case Digests** (~10): URA Case Digest volumes with tax tribunal summaries
- **Compliance Materials** (~50+): AEO manuals, CRS reporting, exchange of information
- **Tax Waivers & Rulings** (~20+): Tax waiver notices, ADR reviews
- **Newsletters & Reports** (~50+): URA newsletters, annual data books

## Data Access

Documents are discovered via the WordPress REST API (`/wp-json/wp/v2/media`) filtered to English-language PDFs. Full text is extracted using the project's common PDF extraction pipeline.
