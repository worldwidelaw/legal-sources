# Uganda Legal Information Institute

**Source:** [https://ulii.org/](https://ulii.org/)
**Country:** UG
**Data types:** case_law, legislation
**Status:** Blocked

## Why this source is blocked

**Category:** Cloudflare protection

**Technical reason:** `cloudflare_and_tos`

**Details:** Cloudflare JS challenge on individual doc pages. API exists but returns 403. ToS explicitly prohibits scraping and bulk downloads. Laws.Africa API covers legislation only and requires auth token. robots.txt blocks AI bots entirely.

## How you can help

- A residential proxy could bypass datacenter IP blocks
