**Source:** [https://mvlaw.gov.mv/](https://mvlaw.gov.mv/)

**Data types:** legislation

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `auth_required`

**Details:** API at /api/v1 requires OAuth 2.0 client credentials. No public registration available. Web interface is JS-rendered SPA. Would need to contact Attorney General Office for API credentials.
