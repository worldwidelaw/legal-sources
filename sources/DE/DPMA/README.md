# German Patent Office (DPMA)

**Source:** [https://www.dpma.de](https://www.dpma.de)
**Country:** DE
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** no_free_case_law_access

**Technical reason:** `no_free_case_law_access`

**Details:** DPMA decisions (Beschlüsse) are not published openly. The BlPMZ journal containing court decisions is behind the Wolters Kluwer paywall. Patentblatt only has registration notices. BPatG (Federal Patent Court) decisions are a separate source.
