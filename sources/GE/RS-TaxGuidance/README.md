# Georgia Revenue Service Tax Guidance (rs.ge)

**Source:** [https://www.rs.ge/home-en](https://www.rs.ge/home-en)
**Country:** GE
**Data types:** doctrine
**Status:** Blocked

## Why this source is blocked

**Category:** JavaScript SPA (single-page application)

**Technical reason:** `no_api_no_structured_data`

**Details:** No public API for tax guidance. rs.ge only provides waybill/invoicing APIs (auth required). Tax guidance scattered as PDFs and FAQs. InfoHub portal requires registration.
