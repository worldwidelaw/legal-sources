## Why this source is blocked

**Category:** API key required

**Technical reason:** `requires_api_key`

**Details:** Laws.Africa Content API returns 403 without auth token. Same blocker as INTL/LawsAfrica. NigeriaLII has 0 legislation documents. Admin needs to sign up at edit.laws.africa.
