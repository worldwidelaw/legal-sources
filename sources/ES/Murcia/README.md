# ES/Murcia — Murcia Regional Legislation (BORM)

Regional legislation of the Comunidad Autónoma de la Región de Murcia, sourced from
the BOE (Boletín Oficial del Estado) via ELI sitemaps and the Open Data API.

## Coverage

- **Temporal**: 1982–present
- **Document types**: Leyes, Decretos, Acuerdos, Órdenes, Reglamentos, Resoluciones
- **Estimated count**: ~434 documents
- **Language**: Spanish (es)

## Data Access

1. BOE ELI sitemaps (`sitemap0.xml`–`sitemap2.xml`) with `es-mc` prefix
2. BOE Open Data API for metadata (JSON) and full text (XML)
