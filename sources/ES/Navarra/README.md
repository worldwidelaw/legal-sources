# ES/Navarra — Navarra Regional Legislation (BON)

Regional legislation of the Comunidad Foral de Navarra, sourced from
the BOE (Boletín Oficial del Estado) via ELI sitemaps and the Open Data API.

Navarra has a distinctive foral civil law tradition with specific document types
(Leyes Forales, Decretos Forales, etc.).

## Coverage

- **Temporal**: 1982–present
- **Document types**: Leyes Forales, Decretos Forales, Decretos Forales Legislativos, Decretos Ley Forales, Órdenes Forales, Resoluciones, Acuerdos
- **Estimated count**: ~1175 documents
- **Language**: Spanish (es)

## Data Access

1. BOE ELI sitemaps (`sitemap0.xml`–`sitemap2.xml`) with `es-nc` prefix
2. BOE Open Data API for metadata (JSON) and full text (XML)
