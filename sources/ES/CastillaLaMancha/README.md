# ES/CastillaLaMancha — Castilla-La Mancha Regional Legislation (DOCM)

Regional legislation from the Diario Oficial de Castilla-La Mancha (DOCM),
the official gazette of the autonomous community of Castilla-La Mancha, Spain.

## Data Source

- **Publisher:** Junta de Comunidades de Castilla-La Mancha
- **URL:** https://docm.jccm.es/
- **Access method:** ELI (European Legislation Identifier) hierarchy
- **Coverage:** 1983–present
- **Document types:** Laws, decrees, legislative decrees, orders, resolutions,
  agreements, instructions, regulations
- **Language:** Spanish (es)
- **Full text:** Yes — structured HTML via ELI endpoints

## Usage

```bash
python bootstrap.py test                  # Connectivity test
python bootstrap.py bootstrap --sample    # Fetch sample records
python bootstrap.py bootstrap             # Full bootstrap
python bootstrap.py update                # Incremental update
```
