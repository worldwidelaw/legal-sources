# Spanish Provincial Official Gazettes (BOPs)

**Source:** [https://bop.dipualba.es/](https://bop.dipualba.es/)
**Country:** ES
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** requires_41_individual_scrapers

**Technical reason:** `requires_41_individual_scrapers`

**Details:** No centralized API. 41 provinces each with unique website/CMS. Would require 41+ individual scrapers. Some provinces have CKAN open data (Huesca), but most are HTML-only. datos.gob.es has no aggregated dataset. Scope too large for single-source framework.
