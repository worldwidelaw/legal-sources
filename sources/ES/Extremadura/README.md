# ES/Extremadura — Extremadura Regional Legislation (DOE)

Regional legislation from the Diario Oficial de Extremadura (DOE), the official journal of
the autonomous community of Extremadura, Spain.

## Data Source

- **Publisher:** Junta de Extremadura
- **URL:** https://doe.juntaex.es/
- **Coverage:** 1984–present
- **Language:** Spanish (es)
- **Document types:** Leyes, Decretos, Decretos-Ley, Decretos Legislativos, Órdenes, Reglamentos

## Methodology

1. Navigate ELI (European Legislation Identifier) hierarchy to discover documents
2. Resolve XML document IDs via ELI PDF reference endpoints
3. Fetch full text as structured XML from DOE servers
