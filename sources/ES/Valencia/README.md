# ES/Valencia — Valencia Regional Legislation (DOGV)

Regional legislation of the Comunitat Valenciana, sourced from
the BOE (Boletín Oficial del Estado) via ELI sitemaps and the Open Data API.

## Coverage

- **Temporal**: 1982–present
- **Document types**: Leyes, Decretos, Decretos Ley, Órdenes, Reglamentos, Resoluciones, Acuerdos
- **Estimated count**: ~610 documents
- **Language**: Spanish (es), Valencian/Catalan

## Data Access

1. BOE ELI sitemaps (`sitemap0.xml`–`sitemap2.xml`) with `es-vc` prefix
2. BOE Open Data API for metadata (JSON) and full text (XML)
