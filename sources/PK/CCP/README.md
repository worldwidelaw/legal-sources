## Why this source is blocked

**Category:** Scanned PDFs requiring OCR

**Technical reason:** `scanned_pdf_no_ocr`

**Details:** All 171 order PDFs are scanned images (CCITTFaxDecode) without OCR text layer. No text extraction possible.
