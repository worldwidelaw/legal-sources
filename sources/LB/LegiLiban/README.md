# LegiLiban Legal Informatics Center

**Source:** [http://legiliban.ul.edu.lb](http://legiliban.ul.edu.lb)
**Country:** LB
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** institutional_access_only

**Technical reason:** `institutional_access_only`

**Details:** From issue #228. Lebanese University Center for Legal Informatics. Contains all Lebanese legislation since Ottoman period (1920+). Only accessible on-site at Lebanese university libraries. Connection refused from external IPs.
