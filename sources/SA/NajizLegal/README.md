# Saudi Arabia Najiz Judicial Portal

**Source:** [https://www.moj.gov.sa/](https://www.moj.gov.sa/)
**Country:** SA
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `auth_required`

**Details:** API gateway at laws-gateway.moj.gov.sa requires JWT auth. Developer registration at developers.najiz.sa requires organizational account (government/company). 151 APIs available but all gated behind auth subscription.

## How you can help

This source requires login credentials or API keys.
