# Korea Legislation Research Institute English Law API

**Source:** [https://openlaw.klri.re.kr/service/apiguide/apiguideinfo.do](https://openlaw.klri.re.kr/service/apiguide/apiguideinfo.do)
**Country:** KR
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `auth_required`

**Details:** API requires registered user ID (email) with approval. Test confirmed id=admin returns 'unregistered ID' error. Cannot use without sign-up and approval.

## How you can help

This source requires login credentials or API keys.
- If this data is publicly available through another channel, please file an issue
- If you have legitimate access, you could contribute a data dump

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
