# Korea Open Law Data API (data.go.kr)

**Source:** [http://apis.data.go.kr/1170000/law](http://apis.data.go.kr/1170000/law)
**Country:** KR
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** API key required

**Technical reason:** `needs_api_key`

**Details:** Requires data.go.kr API key registration. Duplicate of KR/LawGoKr which covers same legislation without auth.

## How you can help

The API requires a key that must be obtained from the provider.
- If you have a key or know how to get one freely, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
