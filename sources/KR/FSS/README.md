# Korea Financial Supervisory Service (FSS)

**Source:** [https://www.fss.or.kr/](https://www.fss.or.kr/)
**Country:** KR
**Data types:** doctrine
**Status:** Blocked

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `auth_required`

**Details:** OPENDART API at opendart.fss.or.kr provides corporate filings (XBRL/XML) but requires free API key registration. FSS enforcement actions and regulatory guidance have no API — web pages and PDFs only.
