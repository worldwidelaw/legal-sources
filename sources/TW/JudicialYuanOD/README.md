# Taiwan Judicial Yuan Open Data Platform

**Source:** [https://opendata.judicial.gov.tw/](https://opendata.judicial.gov.tw/)
**Country:** TW
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** API key required

**Technical reason:** `needs_token`

**Details:** API updated Aug ount + token auth (6-hour expiry). Monthly archives behind Cloudflare Turnstile. FJUD search requires ASP.NET session. Admin must register at opendata.judicial.gov.tw to get API credentials.

## How you can help

The API requires a key that must be obtained from the provider.
- If you have a key or know how to get one freely, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
