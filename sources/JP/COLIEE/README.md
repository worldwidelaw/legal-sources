# COLIEE Japanese Legal Corpus

**Source:** [https://sites.ualberta.ca/~rabelo/COLIEE2024/](https://sites.ualberta.ca/~rabelo/COLIEE2024/)
**Country:** JP
**Data types:** case_law, legislation
**Status:** Blocked

## Why this source is blocked

**Category:** Registration required

**Technical reason:** `registration_required`

**Details:** Academic competition dataset requiring formal application and signed memorandums. Not open data. Data from Federal Court of Canada (Compass Law) and Japanese Civil Code.

## How you can help

Access requires creating an account or registering for an API key.
- If the registration is free and the data is open, we may revisit this

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
