# NZ/ERA — Employment Relations Authority Determinations

Employment Relations Authority (ERA) determinations from New Zealand, covering
employment disputes since 2000. Approximately 21,000 decisions.

- **Source**: https://determinations.era.govt.nz/determinations
- **Data type**: case_law
- **Coverage**: 2000–present (~21,000 determinations)
- **Full text**: Extracted from PDFs (available since 2005)
- **Strategy**: Sequential ID enumeration of determination pages, parse metadata
  from HTML, download and extract PDF full text.
