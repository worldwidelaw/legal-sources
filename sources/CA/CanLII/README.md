# CanLII (Canadian Legal Information Institute)

**Source:** [https://www.canlii.org](https://www.canlii.org)
**Country:** CA
**Data types:** case_law, legislation
**Status:** Blocked

## Why this source is blocked

**Category:** Unknown

**Details:** BLOCKED: API key required (apply via feedback form). API only provides metadata, not full text. Citation graph data is unique but not sufficient alone. Admin needs to apply for key.
