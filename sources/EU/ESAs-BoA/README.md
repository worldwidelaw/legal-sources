# EU/ESAs-BoA — Joint Board of Appeal of the European Supervisory Authorities

Appeals body for the three European Supervisory Authorities: EBA, EIOPA, and ESMA.
Handles appeals against supervisory decisions under EU financial regulation (CRA Regulation,
banking, insurance, securities markets).

- **Coverage:** ~26 decisions (2013–present)
- **Language:** English
- **Data type:** case_law
- **Source page:** https://www.eiopa.europa.eu/decisions-board-appeal_en
- **Access method:** HTML listing → PDF download → text extraction
