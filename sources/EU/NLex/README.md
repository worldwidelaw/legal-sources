# N-Lex (EU National Law Portal)

**Source:** [https://n-lex.europa.eu/](https://n-lex.europa.eu/)
**Country:** EU
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** gateway_only_no_data

**Technical reason:** `gateway_only_no_data`

**Details:** N-Lex is a pure search gateway that redirects to national databases. It contains no documents, has no API, no bulk downloads, no SPARQL. National legislation is already covered by per-country sources.
