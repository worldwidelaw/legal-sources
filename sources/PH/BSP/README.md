# PH/BSP — Bangko Sentral ng Pilipinas Circulars

Regulatory issuances from the Bangko Sentral ng Pilipinas (Central Bank of the Philippines), including circulars, memoranda, and circular letters.

## Data Source

- **Website:** https://www.bsp.gov.ph/SitePages/Regulations/RegulationsList.aspx
- **API:** SharePoint REST API (`/_api/Web/Lists/.../Items`)
- **Format:** OData JSON + PDF documents
- **Auth:** None required
- **Coverage:** ~5,100+ issuances (circulars, memoranda, circular letters)

## Data Types

- `doctrine` — regulatory circulars, memoranda, and guidance documents
