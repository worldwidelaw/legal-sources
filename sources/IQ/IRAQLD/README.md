## Why this source is blocked

**Category:** Cloudflare protection

**Technical reason:** `cloudflare_protection`

**Details:** Cloudflare managed challenge blocks all programmatic access. No API. ASP.NET Web Forms with ViewState-dependent navigation. 27K+ legal texts 1917-present.
