# Cambodia Law Database

**Source:** [https://www.cambodianlawbox.com/](https://www.cambodianlawbox.com/)
**Country:** KH
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `paywall`

**Details:** Actual domain is camlawbox.com. 8K+ entries but full text requires paid subscription. Listing pages public, documents behind login. Alternative ODC (CKAN) has PDFs but they are scanned images needing OCR.
