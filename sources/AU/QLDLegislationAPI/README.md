# Queensland Legislation API (Official REST API)

**Source:** [https://www.legislation.qld.gov.au/api](https://www.legislation.qld.gov.au/api)
**Country:** AU
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** Registration required

**Technical reason:** `requires_free_registration`

**Details:** API requires free account registration at api.legislation.qld.gov.au/api/signup (username/password → JWT auth). AustLII QLD databases return 410 Gone. Admin must register and provide credentials.

## How you can help

Access requires creating an account or registering for an API key.
- If the registration is free and the data is open, we may revisit this

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
