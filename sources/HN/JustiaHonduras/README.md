# Honduras Legislation (Justia Mirror)

**Source:** [https://honduras.justia.com/federales/leyes/](https://honduras.justia.com/federales/leyes/)
**Country:** HN
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** Scanned PDFs requiring OCR

**Technical reason:** `scanned_pdf_no_ocr`

**Details:** PDFs are scanned images with no text layer. All tested documents return 0 chars from pdfplumber/pypdf. Requires OCR infrastructure.

## How you can help

Documents are scanned images in PDF format, requiring OCR.
- Tesseract OCR integration would be needed
- If you have OCR expertise, a PR would be welcome
