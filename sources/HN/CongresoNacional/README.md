## Why this source is blocked

**Category:** no_legislation_database

**Technical reason:** `no_legislation_database`

**Details:** congresonacional.hn is a news/communications portal only, not a legislation database. Official source CEDIJ (poderjudicial.gob.hn) exists but server is completely unreachable (timeouts).
