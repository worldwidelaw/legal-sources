# Italian Ministry of Justice Banca Dati

**Source:** [https://bdp.giustizia.it](https://bdp.giustizia.it)
**Country:** IT
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `authentication_required`

**Details:** BDP = Banca Dati Pubblica (3.5M civil court decisions, 2016+). Requires Italian SPID/CIE/CNS digital identity login. No public API, no open data endpoint. See issue #530.

This source requires login credentials or API keys.
