# QA/QFCRT — Qatar International Court and Dispute Resolution Centre (QICDRC)

Judgments from the QICDRC, including the Civil and Commercial Court (First Instance
and Appellate Division) and the Regulatory Tribunal. English-language common-law
court within the Qatar Financial Centre.

- **Coverage:** ~492 judgments (2010–present)
- **Language:** English
- **Format:** PDF decisions, metadata from HTML detail pages
- **Access:** Sitemap discovery → HTML scraping → PDF download + text extraction

## Data Fields

| Field | Description |
|-------|-------------|
| title | Case name (e.g., "Claimant v Respondent") |
| text | Full judgment text extracted from PDF |
| date | Judgment date (ISO 8601) |
| case_number | Official case number |
| neutral_citation | Neutral citation (e.g., [2025] QIC (RT) 1) |
| court_type | Court of First Instance / Appellate Division / Regulatory Tribunal |
| judges | Panel of judges |
| keywords | Subject matter keywords |
| summary | AI-generated summary from QICDRC (where available) |
