# CL/DT — Dirección del Trabajo - Jurisprudencia Administrativa

Chilean Labor Authority administrative jurisprudence: binding dictámenes
(interpretive rulings) and ordinarios (administrative opinions) on
employment law, labor code interpretation, and workers' rights.

- **Documents:** ~6,500 dictámenes and ordinarios (1919-present)
- **Source:** DT Normativa portal (HTML scraping)
- **Full text:** Yes (extracted from article body HTML)
- **Language:** Spanish
- **Update frequency:** Ongoing (new rulings published regularly)
