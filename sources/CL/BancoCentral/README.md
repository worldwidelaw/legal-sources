# CL/BancoCentral — Banco Central de Chile Normativa

Central Bank of Chile regulations including Board Agreements (Acuerdos),
Circulars (Circulares), Circular Letters (Cartas Circulares), Notices
(Avisos), and Certificates (Certificados). Covers monetary policy,
financial regulation, international exchange norms, and prudential standards.

- **Documents:** ~3,400 normative acts
- **Source:** BCN LeyChile API filtered by organismo=Banco Central
- **Full text:** Yes (via get_norma_json endpoint)
- **Language:** Spanish
- **Update frequency:** Ongoing (new norms published regularly)
