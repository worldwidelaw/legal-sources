# CL/SUSESO — Superintendencia de Seguridad Social

Administrative rulings (dictámenes) from Chile's Social Security Superintendent (SUSESO), covering interpretation of social security legislation including workplace accidents (Ley 16.744), occupational diseases, medical leave, disability subsidies, pensions, and family allowances.

- **Records**: ~16,000 dictámenes (1988–present)
- **Type**: case_law (administrative rulings)
- **Language**: Spanish
- **Auth**: None (open access)
- **API**: JSONP search via Newtenberg Engine + HTML article pages
