# CL/TribunalesAmbientales

**Tribunales Ambientales - Chilean Environmental Court Rulings**

Fetches sentencias (rulings) from Chile's three Environmental Courts:

- **1TA** – Primer Tribunal Ambiental (Antofagasta) — currently inaccessible (SSL)
- **2TA** – Segundo Tribunal Ambiental (Santiago) — via WordPress REST API
- **3TA** – Tercer Tribunal Ambiental (Valdivia) — via sentencias HTML page

## Coverage

- ~260 sentencias (2013–2026)
- Full text extracted from PDF documents
- Environmental damage, illegality claims, sanctions appeals

## Data Access

- 3TA: HTML table at `https://3ta.cl/sentencias/` with links to sentencia PDFs
- 2TA: WordPress API news posts containing PDF links
- 1TA: WordPress site at `www.1ta.cl` (blocked by TLS version incompatibility)
