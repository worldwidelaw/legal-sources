# Chile Poder Judicial Portal Unificado de Sentencias

**Source:** [https://www.pjud.cl/portal-unificado-sentencias](https://www.pjud.cl/portal-unificado-sentencias)
**Country:** CL
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** CAPTCHA protection

**Technical reason:** `recaptcha_v3`

**Details:** Portal Unificado is a landing page linking to juris.pjud.cl which has reCAPTCHA v3 on every page. Transparency AJAX endpoints blocked by F5 WAF. Statistics API has only aggregates, no individual judgments. Same blocker as CL/PoderJudicial.

## How you can help

The site requires solving CAPTCHAs to access content.
