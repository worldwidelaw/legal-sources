# CL/UAF — Unidad de Análisis Financiero

Chile's Financial Analysis Unit. Publishes anti-money laundering (AML) and counter-terrorism financing (CFT) circulars under Ley 19.913.

## Coverage

- **Active circulars**: N°36, N°50, N°61, N°62 (current AML/CFT instructions)
- **Derogated circulars**: N°6–N°60 (~24 repealed circulars)
- **Law**: Ley 19.913 (Spanish + English versions)
- **Policy documents**: Supervision and sanctions policies
- **Catalog**: Predicate offences for money laundering
