# Belgian Finance Ministry Tax Circulars

**Source:** [https://eservices.minfin.fgov.be](https://eservices.minfin.fgov.be)
**Country:** BE
**Data types:** doctrine
**Status:** Blocked

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `auth_required`

**Details:** Fisconetplus (tax circulars database) requires SharePoint authentication. No public API available. eservices.minfin.fgov.be only provides TARBEL tariff browser, not circulars.
