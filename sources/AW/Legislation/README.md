# Aruba Government Legislation (Wetgeving)

**Source:** [https://www.gobierno.aw/nl/wetgeving-0](https://www.gobierno.aw/nl/wetgeving-0)
**Country:** AW
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** JavaScript SPA (single-page application)

**Technical reason:** `no_api_available`

**Details:** Aruba domestic legislation (landsverordeningen) is NOT on CVDR/lokaleregelgeving.overheid.nl (only CW and SX are). gobierno.aw has legislation categories but only PDF downloads with no API. No structured data endpoint found.
