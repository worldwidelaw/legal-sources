# Diário Oficial da União (Brazilian Federal Official Gazette)

**Source:** [https://www.in.gov.br/](https://www.in.gov.br/)
**Country:** BR
**Data types:** legislation
**Status:** Blocked

## Why this source is blocked

**Category:** site_blocks_automated_access

**Technical reason:** `site_blocks_automated_access`

**Details:** in.gov.br drops connections for automated requests (RemoteDisconnected). INLABS bulk XML requires registration (email/password auth). dados.gov.br API also requires login. No unauthenticated programmatic access available.
