# Maranhão State Court (TJMA) Jurisprudence

**Source:** [https://www.tjma.jus.br/](https://www.tjma.jus.br/)
**Country:** BR
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** CAPTCHA protection

**Technical reason:** `geo_restricted_captcha`

**Details:** Jurisprudence API at apijuris.tjma.jus.br is geo-restricted (timeout from outside Brazil) and uses CAPTCHA. Custom Angular/Laravel system, not eSAJ.

## How you can help

The site requires solving CAPTCHAs to access content.
- If you know of an alternative API or bulk download for this data, please let us know
