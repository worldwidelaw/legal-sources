# CADE - Brazilian Competition Authority Jurisprudence

**Source:** [https://jurisprudencia.cade.gov.br/](https://jurisprudencia.cade.gov.br/)
**Country:** BR
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `bot_protection_and_auth_required`

**Details:** Jurisprudence portal uses F5/Shape Security bot protection + gov.br login. REST API (apis.cade.gov.br) requires ConnectaGov OAuth2 registration and only does CPF/CNPJ lookups, not full-text. SEI portal has reCAPTCHA after 5 searches. Contact apis@cade.gov.br for bulk access.
