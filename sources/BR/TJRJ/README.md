# TJRJ - Rio de Janeiro State Court Jurisprudence

**Source:** [https://www3.tjrj.jus.br/ejuris/ConsultarJurisprudencia.aspx](https://www3.tjrj.jus.br/ejuris/ConsultarJurisprudencia.aspx)
**Country:** BR
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** CAPTCHA protection

**Technical reason:** `captcha_required`

**Details:** All full-text endpoints (eJURIS, eJUD, ConsultarEmentarios) protected by reCAPTCHA v3 (server-validated). DJERJ gazette only serves Administrative + Editais cadernos (no judicial decisions). DataJud API has metadata only (no decision text). eproc returns 403. Requires browser automation to solve CAPTCHA.
