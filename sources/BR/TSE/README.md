# TSE - Tribunal Superior Eleitoral (Electoral Court Open Data)

**Source:** [https://dadosabertos.tse.jus.br/](https://dadosabertos.tse.jus.br/)
**Country:** BR
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** CAPTCHA protection

**Technical reason:** `captcha_required`

**Details:** Search API at sjur-pesquisa-api.tse.jus.br requires hCaptcha from external networks. DataJud API has metadata only (no full text). Open data CSVs are metadata-only.

## How you can help

The site requires solving CAPTCHAs to access content.
- If you know of an alternative API or bulk download for this data, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
