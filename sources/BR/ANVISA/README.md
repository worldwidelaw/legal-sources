## Why this source is blocked

**Category:** site_blocks_automated_access

**Technical reason:** `site_blocks_automated_access`

**Details:** AnvisaLegis (datalegis.net) returns 403 via Cloudflare, robots.txt blocks bots. DOU (in.gov.br) drops connections for automated requests (same as BR/DOU). No unauthenticated programmatic access to ANVISA regulatory texts.
