# Open Courts Slovakia (Otvorené súdy)

**Source:** [https://otvorenesudy.sk](https://otvorenesudy.sk)
**Country:** SK
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** API key required

**Technical reason:** `api_key_required`

**Details:** API at api.otvorenesudy.sk requires invite-only API key. Request at api@otvorenesudy.sk. Official justice portal (obcan.justice.sk) has no open API either. Admin must request API key to unblock.

## How you can help

The API requires a key that must be obtained from the provider.
- If you have a key or know how to get one freely, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
