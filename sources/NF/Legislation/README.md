## Why this source is blocked

**Category:** database_empty

**Technical reason:** `database_empty`

**Details:** AustLII NF database returns HTTP 500 on all paths and 0 search results. Norfolk Island lost self-governance in 2015; its legislation was never uploaded to AustLII. Only Commonwealth acts about NF exist (under CTH). No alternative open source found.
