# Russia Arbitrazh Courts Database (kad.arbitr.ru)

**Source:** [https://kad.arbitr.ru/](https://kad.arbitr.ru/)
**Country:** RU
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** Geographic restriction

**Technical reason:** `anti_bot_and_geoblocking`

**Details:** kad.arbitr.ru returns HTTP 451 from non-Russian IPs, has custom CAPTCHA (PravoCaptcha), aggressive rate limiting, and IP blocking. No open data alternative exists for Russian arbitrazh court decisions.
