# Liechtenstein — Legal Data Sources

> **Last updated:** 2026-02-21

## Overview

Liechtenstein is a **constitutional monarchy** with a **civil law** system (Austrian/Swiss influence). Provides **consolidated legislation** through **LILEX** (gesetze.li, from 1921+, German). Constitutional Court (StGH) decisions also indexed.
