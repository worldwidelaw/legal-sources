# Kazakhstan Supreme Court Decisions

**Source:** [https://sud.gov.kz/](https://sud.gov.kz/)
**Country:** KZ
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** Registration required

**Technical reason:** `registration_and_digital_signature_required`

**Details:** No public API. Full text at sud.gov.kz and office.sud.kz requires registered account with digital signature (EDS). adilet.zan.kz (legislation) also has no API. data.egov.kz open data portal has no court decision datasets.

## How you can help

Access requires creating an account or registering for an API key.
- If the registration is free and the data is open, we may revisit this

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
