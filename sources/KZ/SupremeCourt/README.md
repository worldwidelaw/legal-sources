# Kazakhstan Supreme Court Judicial Database

**Source:** [https://sud.gov.kz/](https://sud.gov.kz/)
**Country:** KZ
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** CAPTCHA protection

**Technical reason:** `captcha_required`

**Details:** Both sud.gov.kz/rus/bank and office.sud.kz/courtActs require Google reCAPTCHA to search. No public API, RSS feed, or open data endpoint for court decisions. data.egov.kz has no court decisions dataset.

## How you can help

The site requires solving CAPTCHAs to access content.
- If you know of an alternative API or bulk download for this data, please let us know

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
