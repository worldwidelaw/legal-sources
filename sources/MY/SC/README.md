# MY/SC — Securities Commission Malaysia Enforcement

Enforcement actions from the Securities Commission Malaysia (sc.com.my).

Covers criminal prosecutions (1993-present), civil actions & regulatory settlements (2005-present), cases compounded (1996-present), and administrative actions (2002-present).

## Data

- **Type:** case_law, doctrine
- **Format:** HTML (scraped from enforcement tables)
- **Language:** English
- **Records:** ~15+ enforcement actions with full text
