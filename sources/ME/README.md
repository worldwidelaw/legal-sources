# Montenegro — Legal Data Sources

> **Last updated:** 2026-02-21

## Overview

Montenegro is a **civil law** country, **unitary**, EU candidate. Publishes through the **Sluzbeni list CG** (official gazette). Constitutional Court decisions are indexed.

## Consolidated Legislation vs. Official Journal

Montenegro does **not** have a consolidated legislation database. The Sluzbeni list publishes laws as enacted.

## Sub-jurisdictions

**Unitary state** — 25 municipalities with no legislative autonomy.
