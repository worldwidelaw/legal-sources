# Serbia — Legal Data Sources

> **Last updated:** 2026-02-21

## Overview

Serbia is a **civil law** country, **unitary** (with autonomous province of Vojvodina; Kosovo* status disputed). Publishes through the **Sluzbeni glasnik RS** (official gazette).

## Consolidated Legislation vs. Official Journal

Serbia does **not** have an official consolidated legislation database. The Sluzbeni glasnik publishes laws as enacted.

## Sub-jurisdictions

Vojvodina has an autonomous provincial assembly but limited legislative autonomy.

Priority: Constitutional Court, Supreme Court, NBS, competition authority. Create directories under `sources/RS/[SourceName]/`.
