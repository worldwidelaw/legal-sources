# Iran Supreme Court Opinions (Divanealee)

**Source:** [https://www.divanealee.ir/](https://www.divanealee.ir/)
**Country:** IR
**Data types:** case_law
**Status:** Blocked

## Why this source is blocked

**Category:** Geographic restriction

**Technical reason:** `geo_blocked`

**Details:** All Iranian legal databases geo-blocked from outside Iran (403/timeout). No APIs exist. Domain divanealee.ir defunct, migrated to eadl.ir which is also blocked.
