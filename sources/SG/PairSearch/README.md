# Singapore PairSearch (Hansard & Supreme Court)

**Source:** [https://search.pair.gov.sg/](https://search.pair.gov.sg/)
**Country:** SG
**Data types:** case_law, doctrine
**Status:** Blocked

## Why this source is blocked

**Category:** Authentication required

**Technical reason:** `auth_required`

**Details:** PairSearch API requires session authentication (401 on all data endpoints). No public API key or token mechanism. Underlying data available via SPRS (sprs.parl.gov.sg) for Hansard and eLitigation for judiciary - these are separate sources.

## How you can help

This source requires login credentials or API keys.
- If this data is publicly available through another channel, please file an issue
- If you have legitimate access, you could contribute a data dump

- File an issue or open a PR at [worldwidelaw/legal-sources](https://github.com/worldwidelaw/legal-sources)
